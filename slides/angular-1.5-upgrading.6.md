Migration scenarios
-------
* **Incremental**
  1. Convert syntax (ng-forward), 
  2. Upgrade components to ng 2 (ng-forward + ng-upgrade)
  3. Remove ng-forward (ng-upgrade)
  4. Upgrade everything else (Angular 2)
* **Small bang**
  1. Convert syntax
  2. Upgrade everything
* **Big bang**
  1. Upgrade everything
